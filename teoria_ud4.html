Unidad 4: Programación Orientada a Objetos en Java

Clases, objetos, herencia, polimorfismo y encapsulamiento.

1. Conceptos de POO

  • Clase: Plantilla para crear objetos.
  • Objeto: Instancia de una clase.
  • Herencia: Una clase puede heredar atributos y métodos de otra.
  • Polimorfismo: Un mismo método puede comportarse de diferentes formas.
  • Encapsulamiento: Ocultar detalles internos mediante modificadores de acceso (private, public).

2. Ejemplo de clase en Java

public class Persona {
    private String nombre;
    private int edad;

    public Persona(String nombre, int edad) {
        this.nombre = nombre;
        this.edad = edad;
    }

    public void saludar() {
        System.out.println("Hola, soy " + nombre);
    }
}

3. Herencia

public class Estudiante extends Persona {
    private String carrera;

    public Estudiante(String nombre, int edad, String carrera) {
        super(nombre, edad);
        this.carrera = carrera;
    }
}
