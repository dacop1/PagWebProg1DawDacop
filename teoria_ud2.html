Unidad 2: Arrays y listas

Estructuras de datos: listas, arrays y matrices en Python y Java.

1. ¿Qué es un array?

Un array (o lista) es una estructura de datos que permite almacenar una colección de elementos en posiciones contiguas de memoria. En Python, las listas son dinámicas y pueden contener elementos de diferentes tipos.

2. Listas en Python

# Crear una lista
frutas  = ["manzana", "banana", "cereza"]
numeros = [1, 2, 3, 4, 5]
mixta   = [10, "hola", 3.14, True]

3. Operaciones comunes

  • Acceder a elementos: frutas[0] → "manzana"
  • Modificar: frutas[1] = "naranja"
  • Añadir al final: frutas.append("uva")
  • Eliminar: frutas.remove("manzana") o del frutas[0]
  • Longitud: len(frutas)

4. Iterar sobre listas

for fruta in frutas:
    print(fruta)

for i in range(len(frutas)):
    print(frutas[i])

5. Métodos útiles

  • sort(): ordena la lista.
  • reverse(): invierte el orden.
  • index(elemento): devuelve la posición de un elemento.
  • count(elemento): cuenta cuántas veces aparece.

6. Arrays en Java

En Java, los arrays tienen tamaño fijo y se declaran con el tipo:

int[] numeros = new int[5];
numeros[0] = 10;

String[] nombres = {"Ana", "Luis"};

// Recorrer
for (int i = 0; i < nombres.length; i++) {
    System.out.println(nombres[i]);
}
