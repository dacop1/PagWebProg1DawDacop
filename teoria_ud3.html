Unidad 3: Introducción a Java

Sintaxis básica, tipos de datos y estructura de un programa Java.

1. ¿Qué es Java?

Java es un lenguaje de programación orientado a objetos, compilado y multiplataforma gracias a la máquina virtual de Java (JVM). Fue desarrollado por Sun Microsystems (ahora Oracle) y es uno de los lenguajes más utilizados en el mundo.

2. Características principales

  • Orientado a objetos: Todo en Java (excepto tipos primitivos) son objetos.
  • Independiente de la plataforma: El código compilado (bytecode) se ejecuta en cualquier sistema que tenga una JVM.
  • Recolección automática de memoria: El programador no necesita gestionar la memoria manualmente.
  • Seguro y robusto: Tiene mecanismos que evitan accesos indebidos a memoria.

3. Estructura de un programa Java

Todo programa Java debe tener al menos una clase y un método main que es el punto de entrada.

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola, mundo!");
    }
}

Explicación:

  • public class HolaMundo → Define una clase pública llamada HolaMundo.
  • public static void main(String[] args) → Método principal.
  • System.out.println() → Imprime en consola.

4. Tipos de datos primitivos

  • byte → 8 bits (-128 a 127)
  • short → 16 bits (-32,768 a 32,767)
  • int → 32 bits (entero típico)
  • long → 64 bits (requiere sufijo L)
  • float → 32 bits (decimal, sufijo f)
  • double → 64 bits (decimal)
  • char → 16 bits (caracter Unicode, comillas simples)
  • boolean → true o false

5. Variables y constantes

int edad = 25;
double precio = 19.99;
char letra = 'A';
boolean esMayor = true;

// Constante (final)
final double PI = 3.1416;

6. Estructuras de control

Condicional if-else

int nota = 85;
if (nota >= 60) {
    System.out.println("Aprobado");
} else {
    System.out.println("Reprobado");
}

Switch

int dia = 2;
switch (dia) {
    case 1:
        System.out.println("Lunes");
        break;
    case 2:
        System.out.println("Martes");
        break;
    default:
        System.out.println("Otro día");
}

Bucles

// for
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

// while
int j = 0;
while (j < 5) {
    System.out.println(j);
    j++;
}

7. Arrays

// Declaración e inicialización
int[] numeros = new int[5];
numeros[0] = 10;

// Declaración con valores
String[] nombres = {"Ana", "Luis", "Carlos"};

// Recorrer array
for (int i = 0; i < nombres.length; i++) {
    System.out.println(nombres[i]);
}

8. Entrada y salida básica

Para leer datos del teclado se usa la clase Scanner (del paquete java.util).

import java.util.Scanner;

public class Entrada {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Ingresa tu nombre: ");
        String nombre = sc.nextLine();
        System.out.println("Hola, " + nombre);
        sc.close();
    }
}
