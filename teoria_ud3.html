Unidad 3: Introducción a Java

Sintaxis básica, tipos de datos y estructura de un programa Java.

1. ¿Qué es Java?

Java es un lenguaje de programación orientado a objetos, compilado y multiplataforma gracias a la máquina virtual de Java (JVM).

2. Estructura de un programa Java

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola, mundo!");
    }
}

3. Tipos de datos primitivos

  • int (entero)
  • double (decimal)
  • boolean (true/false)
  • char (carácter)

4. Variables y constantes

int edad = 25;
final double PI = 3.1416;

5. Estructuras de control

Similares a otros lenguajes: if, else, switch, for, while.
