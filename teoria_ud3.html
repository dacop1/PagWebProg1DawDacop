Unidad 3: Introducción a Java

Sintaxis básica, tipos de datos y estructura de un programa Java.

1. ¿Qué es Java?

Java es un lenguaje de programación orientado a objetos, compilado y multiplataforma gracias a la máquina virtual (JVM). Fue desarrollado por Sun Microsystems (ahora Oracle) y es uno de los más utilizados en el mundo.

2. Características principales

  • Orientado a objetos: todo (excepto tipos primitivos) son objetos.
  • Independiente de la plataforma: el bytecode se ejecuta en cualquier sistema con JVM.
  • Gestión automática de memoria: recolector de basura integrado.
  • Seguro y robusto: mecanismos que evitan accesos indebidos a memoria.

3. Estructura de un programa Java

Todo programa Java debe tener al menos una clase y un método main como punto de entrada.

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola, mundo!");
    }
}

4. Tipos de datos primitivos

  • byte → 8 bits (-128 a 127)
  • short → 16 bits
  • int → 32 bits (entero típico)
  • long → 64 bits (sufijo L)
  • float → 32 bits decimal (sufijo f)
  • double → 64 bits decimal
  • char → 16 bits, caracter Unicode
  • boolean→ true o false

5. Variables y constantes

int edad        = 25;
double precio   = 19.99;
char letra      = 'A';
boolean esMayor = true;

final double PI = 3.1416; // Constante

6. Estructuras de control

Condicional if-else

int nota = 85;
if (nota >= 60) {
    System.out.println("Aprobado");
} else {
    System.out.println("Reprobado");
}

Switch

int dia = 2;
switch (dia) {
    case 1: System.out.println("Lunes");  break;
    case 2: System.out.println("Martes"); break;
    default: System.out.println("Otro");
}

Bucles

// for
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

// while
int j = 0;
while (j < 5) {
    System.out.println(j++);
}

7. Arrays en Java

int[] numeros = new int[5];
numeros[0] = 10;

String[] nombres = {"Ana", "Luis", "Carlos"};

for (int i = 0; i < nombres.length; i++) {
    System.out.println(nombres[i]);
}

8. Entrada de datos con Scanner

import java.util.Scanner;

public class Entrada {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Ingresa tu nombre: ");
        String nombre = sc.nextLine();
        System.out.println("Hola, " + nombre);
        sc.close();
    }
}
